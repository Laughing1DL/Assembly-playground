# x86 Assembly-playground

Este repositorio contiene una colección de ejercicios y programas realizados en lenguaje ensamblador x86 (`.286`) utilizando interrupciones de DOS (`int 21h`).

El objetivo principal de la repositorio es practicar conceptos básicos y fundamentales de programación en ensamblador, incluyendo:

- Manejo de registros
- Uso de segmentos (`stack`, `data`, `code`)
- Entrada y salida de datos
- Loops
- Saltos (`jmp`)
- Operaciones matemáticas
- Manejo de caracteres ASCII
- Uso de interrupciones de DOS

Todos los programas fueron desarrollados para ejecutarse en entornos compatibles con DOS, como:

- DOSBox
- MASM
- TASM

---

# Estructura del Repositorio

El contenido está organizado en carpetas con fechas, donde cada carpeta representa ejercicios o prácticas realizadas en diferentes días.

## Carpetas

### `2026-13-4`

Contiene ejercicios básicos relacionados con:

- Impresión de mensajes
- Uso de etiquetas y saltos (`jmp`)
- Flujo de ejecución entre diferentes bloques de código

---

### `2026-17-4`

Incluye programas enfocados en:

- Loops (`loop`)
- Contadores
- Repetición de mensajes
- Entrada de datos desde teclado

---

### `2026-23-4`

Contiene ejercicios de:

- Loops anidados
- Patrones de caracteres
- Impresión de figuras usando `*`
- Manejo de contadores múltiples

---

### `2026-30-2`

Incluye programas relacionados con:

- Operaciones matemáticas
- División
- Manejo de residuos
- Conversión entre ASCII y números

# Tecnologías Utilizadas

- Assembly x86 (.286)
- DOS Interruptions (`int 21h`)
- MASM / TASM
- DOSBox

# Objetivo del Repositorio

Este repositorio funciona como práctica y aprendizaje de programación en bajo nivel utilizando ensamblador x86 y conceptos básicos de arquitectura de computadoras.
