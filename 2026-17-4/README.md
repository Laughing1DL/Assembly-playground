# Programa de Loops en Ensamblador x86 (.286)

Este programa está escrito en lenguaje ensamblador para procesadores Intel 80286 (`.286`) y utiliza interrupciones de DOS (`int 21h`) para interactuar con el usuario.

El objetivo del programa es:

1. Pedir al usuario un número.
2. Guardar ese número como cantidad de repeticiones.
3. Mostrar un mensaje varias veces utilizando la instrucción `loop`.

---

# Estructura General del Programa

El código está dividido en 3 segmentos principales:

- Stack
- Data
- Code
